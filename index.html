How Does It Work?

Install the App

First, get the app from the Play Store / App Store, install it on your device.

Take a Picture

Next, tap on the image box to capture a picture, using your phone's camera.

Run the image through the ML model

The picture clicked gets processed through the TFLite CNN model, and prints out it's prediction along with a confidence score.

Learn about the Clouds

Press the button below to see information on the different types of clouds and what each cloud type signifies.
